# AI 文件維護規則

當這個專案未來繼續改版時，若有以下任一情況，必須同步更新 `docs/AI_CONTINUATION.md` 與 `docs/ai_handoff.json`：

1. 新增大型系統，例如：製作、裝備、天賦、正式主職業、隊伍、城鎮設施。
2. 修改核心規則，例如：戰鬥公式、判定方式、熟練度規則、轉職方式、行動習得技能規則。
3. 資料表新增、刪除、改名。
4. 新增獨立系統檔案或重構系統分工。
5. 起始角色設定、世界觀摘要、內容規模出現重大變動。

## 建議更新順序
1. 先修改實際程式與資料。
2. 確認 `index.html` 可離線執行。
3. 若 `.db` 有變更，重新匯出 `assets/db-cache.js`。
4. 更新 AI 文件兩份。
5. 再重新打包 zip。

## 命名建議
- 每次新版本 zip 檔名維持 `中世紀異世界文字RPG_Beta_vX_X_說明_YYYYMMDD.zip`
- AI 文件檔名盡量固定，不要變動，以利新對話快速定位。
