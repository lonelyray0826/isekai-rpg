灰燼裂隙：幼體求生 Beta v2.7

這是目前可直接接續開發的基底版本。
本版核心方向：重養成 / 生存 / 偏魂類節奏 / 永久死亡 / 方向移動 → 視野刷新 → 物件互動 / 簡潔沉浸式上下版面 UI。

先看：
1. docs/AI_CONTINUATION.md
2. docs/ai_handoff.json
3. assets/origin-system.js
4. assets/perception-system.js
5. assets/combat-system.js

本版最重要變更：
- 介面改為集中式 Focus UI：上方訊息、下方行動、底部收納功能
- 狀態 / 任務 / 敵情 / 地圖 / 技能 / 行囊等資訊改為底部收納切換
- 敘事訊息改為由上往下閱讀，最新內容落在底部
- 新局會隨機抽取出生種族與出身
  - 人類嬰兒：富人家 / 平民家 / 貧民窟
  - 哥布林幼體：群居巢群分支
  - 史萊姆幼體：非群居，具吞食屍體同化能力
- 技能不再機率習得
- 初階技能改為累積「習得熟練」達門檻後自動學會
- 上位技能仍以熟練度滿值後進化
- 成長速度放慢，升級不再輕易增加能力值
- 死亡改為直接遊戲結束，必須重新開局
- 場景操作改為「先看見什麼，再對它做什麼」

- 主行動循環改為：選擇移動方向 → 抵達當前節點 → 視野出現可見物件 → 再決定互動
- 場景新增 `scene_nodes` / `scene_exits`，把「站在哪裡」與「往哪裡挪動」拆開
- `scene_objects` 現在可用 `nodeIds` 限制只在特定視野節點出現
- 主介面行動區會先列出「眼前看見的東西」，再列出「接下來往哪裡動」
- 戰鬥改為「先看見敵人準備什麼，再做反應」
- 內容資料仍以 data/game_content.db 為主，再匯出到 assets/db-cache.js

若後續改版，請同步更新：
- data/game_content.db
- assets/db-cache.js
- docs/AI_CONTINUATION.md
- docs/ai_handoff.json

- 主畫面改成左側「感知與敘事」、右側「當下可做反應」
- 場景與移動按鈕文字改成更貼近角色此刻感知的描述，不再只像抽象系統選單
